Ubuntu 20.04 提示找不到 python
发表于2021-10-25|教程| 条评论
原文地址 blog.csdn.net 问题Ubuntu 20.04 Focal 默认自带 python3.8，但是根据指引，所有引用 python 的包必须显示指定 python3 或其他 python 版本。 Ubuntu 从 20.04 开始不再将 python 加入PATH环境变量，在编译安装一些软件会提示无法运行并提示找不到 python，然而 python3 已安装，需要额外重定向。 解决方法sudo apt install python-is-python3 或者手动创建 symlink sudo ln -s /usr/bin/python3 /usr/bin/python 检查是否配置成功： whereis python
VMware授权服务启动失败、ens33网卡没有IP地址解决方案
发表于2021-10-25|教程| 条评论
VMware授权服务启动失败启动虚拟机报错，根据提示是因为授权服务未能成功启动。 右下角Windows搜索栏直接搜索服务，然后找到VMware Authorization Service，双击设置其启动类型为自动，应用，然后点击启动。 这个时候应该可以启动虚拟机了。 ens33网卡没有IP地址 跟上述原因相同，也有可能是因为VMware DHCP Service没有启动解决办法类似。
