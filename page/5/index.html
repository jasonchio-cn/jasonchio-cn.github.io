VMware授权服务启动失败、ens33网卡没有IP地址解决方案
发表于2021-10-25|教程| 条评论
VMware授权服务启动失败启动虚拟机报错，根据提示是因为授权服务未能成功启动。 右下角Windows搜索栏直接搜索服务，然后找到VMware Authorization Service，双击设置其启动类型为自动，应用，然后点击启动。 这个时候应该可以启动虚拟机了。 ens33网卡没有IP地址 跟上述原因相同，也有可能是因为VMware DHCP Service没有启动解决办法类似。

树莓派（raspberry）启用 root 账户
发表于2021-10-23|教程| 条评论
配置树莓派使用的 linux 是 debian 系统，所以树莓派启用 root 和 debian 是相同的。 debian 里 root 账户默认没有密码，但账户锁定。 当需要 root 权限时，由默认账户经由 sudo 执行，Raspberry pi 系统中的 Raspbian 默认用户是 pi 密码为 raspberry 重新开启 root 账号，可由 pi 用户登录后，在命令行下执行 sudo passwd root 执行此命令后系统会提示输入两遍的 root 密码，输入你想设的密码即可，然后在执行 sudo passwd --unlock root 这样就可以解锁 root 账户了。 参考链接 树莓派（raspberry）启用root账户 - 琴声已散丶弦犹微振 (leif.fun)
