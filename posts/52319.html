条件 对应 container 生命周期 备注
docker container create created
docker container start (created | stopped) –> running
docker container run none –> created –> running
docker container stop running –> stopped
docker container restart running –> running
docker container pause running –> paused
docker container unpause paused –> running
docker container kill running –> stopped
docker container rm (created | stopped) –> deleted
docker container inspect none 获取容器的详细信息
OOM running –> (stopped | running) 当 OOM 时容器会被 kill 掉，
然后根据策略退出或重启。
container process exited running –> (stopped | running) 当容器中的进程退出时，
根据策略进行重启或者 stop。
